ML 10-1 : Déplacement de marqueurs PLOT
à l'aide de la souris dans une fenêtre MATLAB

Versions : MATLAB 4.2c, 5.0, 5.2

Programme MATLAB : DEPLMARK.M

close
clear
axis([0 100 0 100])

% Deplacement du point
cmd1 = [ ...
'pos=get(gca,''currentpoint'');' ...
'set(gco,''xdata'',pos(1,1),''ydata'',pos(1,2));' ...
];

% Reset des callback de la fenetre graphique
cmd2 = [ ...
'set(gcf,''windowbuttonmotionfcn'','''');' ...
'set(gcf,''windowbuttonupfcn'','''');' ...
];

% Affectation des callback de la fenetre graphique
cmd3 = [ ...
'set(gcf,''windowbuttonmotionfcn'',cmd1);' ...
'set(gcf,''windowbuttonupfcn'',cmd2);' ...
];

% Affichage des points
hold on
for i =1:10:100
plot(i,i,'.','markersize',20,'erasemode','xor','buttondownfcn',cmd3);
end

Dans MATLAB, lancez le programme :

>> deplmark
