ML 30-2 : Comment créer des lignes paramétriques dégradées ?

Versions : MATLAB 5.0
Plates-formes : toutes

Question :

Comment générer une ligne 2D ou 3D dont l'amplitude est représentée par les couleurs ?

Réponse

Il est possible de le faire en utilisant la commande mesh. En transformant x, y et/ou z en un tableau 2xN dont la première ligne égale la seconde, il est possible d'utiliser mesh pour tracer une ligne dégradée. Par exemple : 

               % Données originales
               x = 0:pi/5:2*pi; 
               y = sin(x); 
               % Conversion de x et y en tableau 2xN 
               X = [x;x]; 
               Y = [y;y]; 
               % Génération de la ligne dégradée 
               mesh(X,Y,zeros(size(X)),Y) 
               view(2)

Dans cet exemple, une sinusoïde colorée est créée. Les 2 premiers arguments d'entrée du mesh sont les valeurs X et Y. La 3ème entrée est la valeur Z. Si c'est un tracé 2D, la donnée en Z est mise à 0. L'entrée finale est la valeur des couleurs. Si Y représente l'amplitude, mettre les valeurs des couleurs dans Y. Ne pas oublier que, par défaut, si les valeurs des couleurs ne sont pas données, elles correspondent par défaut aux données Z. Comme les données Z sont toutes égales à 0, il faut spécifier les valeurs des couleurs. Pour un tracé 3D, il n'est pas nécessaire de spécifier les valeurs des couleurs.

Vous trouverez ci-dessous un fichier .m qui combine toutes ces étapes dans un seul fichier. Il supporte à la fois des cas 2D et 3D. En utilisant les valeurs de x et y de l'exemple précédent, il est possible de créer le même tracé avec la commande suivante : 

               plotcol(x,y)

Comme vous pouvez le constater, il est plus facile de taper simplement cette ligne. 


               function handle = plotcol(x,y,z,c,linestyle); 
               % PLOTCOL cree un trace qui utilise la palette 
	       % de couleurs
               % 
               % PLOTCOL(X,Y,Z,C,LINESTYLE) trace une ligne
	       % parametrique coloree basee sur X, Y, Z et C
               % en utilisant le style de ligne de LINESTYLE
               % La graduation des couleurs est determinee par
	       % les valeurs de C ou par les valeurs actuelles
	       % de CAXIS.
               % Les valeurs des couleurs graduees sont utilisees
	       % comme indices dans la COLORMAP actuelle
               % 
               % N'importe quelle combinaison d'entrees peut etre 
	       % utilisee. Si C n'est pas donne, il est affecte a
	       % Z, Y ou X selon l'argument d'entree.
               % Ci-dessous se trouve un tableau qui decrit ceci :
               % 
               % Valeur donnee de C 
               % X,Y,Z Z 
               % X,Y Y 
               % X X 
               % 
               % SEE ALSO: mesh 
               if nargin == 0;error('Requires at least one input');end 
               % Determine quels arguments d'entree sont donnes : 
               if nargin == 1; % Trace 2-D, X est donne. 
               [m,n] = size(x); 
               y = [x;x]; 
               x = [1:n;1:n]; 
               z = zeros(2,n); 
               c = y; 
               linestyle = '-'; 
               elseif nargin == 2; % Trace 2-D, X et Y, C, 
               %ou LINESTYLE sont donnes 
               [m,n] = size(x); 
               z = zeros(2,n) 
               if isstr(y) % X et LINESTYLE donnes. 
               y = [x;x]; 
               x = [1:n,1:n]; 
               else % X et Y donnes. 
               x = [x;x]; 
               y = [y;y]; 
               linestyle = '-'; 
               end 
               c = y; 
               elseif nargin == 3; % X, Y, et Z, ou 
               %LINESTYLE donnes. 
               [m,n] = size(x); 
               x = [x;x]; 
               y = [y;y]; 
               if ~isstr(z) % X, Y, et Z donnes 
               z = [z;z]; 
               c = z; 
               linestyle = '-'; 
               else % X, Y, et LINESTYLE 
               % donnes 
               linestyle = z; 
               z = zeros(2,n); 
               c = y; 
               end 
               elseif nargin == 4 % X, Y,  et Z, C, ou 
               % LINESTYLE donnes. 
               [m,n] = size(x); 
               x = [x;x]; 
               y = [y;y]; 
               if isstr(c) % Trace 2-D avec X, Y, 
               % C, et LINESTYLE 
               linestyle = c; 
               c = z; 
               z = zeros(2,n); 
               else % Trace 3-D avec X, Y, 
               % Z, et C ou LINESTYLE 
               linestyle = '-'; 
               z = [z;z]; 
               c = [c;c]; 
               end 
               elseif nargin == 5 % Tout est donne. 
               [m,n] = size(x); 
               x = [x;x]; 
               y = [y;y]; 
               z = [z;z]; 
               c = [c;c]; 
               end 
               h = mesh(x,y,z,c); 
               set(h,'LineStyle',linestyle); 
               if all(z == 0), view(2), end 
               if nargout == 1 
               handle = h; 
               end
