ML 30-3 : Comment placer des labels verticaux sur l'axe des X ?

Versions : MATLAB 5.2
Plates-formes : toutes

Question

Voici un exemple montrant comment faire pivoter les étiquettes de l'axe des abscisses.

Réponse

% Création des données
X = 1:12;
Y = rand(1,12);

% Génération d'un tracé
bar(X,Y);

% Positionnement des tirets sur les axes et suppression 
% des étiquettes
set(gca,'XTick',1:12,'XTickLabel','')

% Définition des étiquettes
lab = ['Jan';'Feb';'Mar';'Apr';'May';'Jun';
       'Jul';'Aug';'Sep';'Oct';'Nov';'Dec'];

% Estimation de la position des étiquettes en se basant 
% sur la position des xlabel
hx = get(gca,'XLabel');  % Handle to xlabel
set(hx,'Units','data');
pos = get(hx,'Position');
y = pos(2);

% Positionnement des nouvelles étiquettes
for i = 1:size(lab,1)
  t(i) = text(i,y,lab(i,:));
end
set(t,'Rotation',90,'HorizontalAlignment','right')
